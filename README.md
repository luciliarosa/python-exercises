# Python Practice Exercises

This repository is dedicated to practicing Python fundamentals through small and progressive exercises.

The goal of this project is to reinforce Python concepts studied in courses and tutorials, especially focusing on backend development and problem-solving skills.

## Purpose

- Practice Python syntax and logic
- Improve understanding of core Python concepts
- Build consistency through daily or frequent exercises
- Support learning from platforms such as Alura
- Track personal progress in Python development

## What you will find here

- Simple Python exercises
- Code examples focused on fundamentals
- Exercises covering topics such as:
  - Variables and data types
  - Functions
  - Conditional statements
  - Loops
  - Lists and dictionaries
  - Error handling
  - Basic input and output

## Project Structure

Each exercise is organized in a clear and simple way, usually as individual `.py` files, making it easy to understand and revisit specific topics.

## How to run the exercises

1. Make sure you have Python installed (version 3.x)
2. Clone this repository
3. Run the desired exercise
